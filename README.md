# sv-idlegame

Bienvenu dans la logique du site vitrine

on a utilise differentes technologies :

1. HTML5
   Structure sémantique
   Canvas pour le trou noir
   Balises modernes
2. CSS3
   Grid & Flexbox → Layout responsive
   Animations CSS → @keyframes, transitions
   Gradients → Dégradés bleu/rose
   Variables CSS → Gestion des couleurs
   media Queries → Responsive design
3. JavaScript Vanilla (sans frameworks)
   DOM Manipulation → querySelector, classList
   Événements → addEventListener
   Canvas API → Animation du trou noir
   Intersection Observer → Animations au scroll
   RequestAnimationFrame → Animations fluides

pour lancer le site il suffit de cliquer sur le fichier index.html
bonne visite a tous!
